# 🎨 CSS Basics

---

## 📌 CSS কী? (What is CSS?)
**CSS**-এর পূর্ণরূপ হলো **Cascading Style Sheets**।

* **Style Sheets:** এটি দিয়ে একটি ওয়েব পেজের চেহারা কেমন হবে তা নির্ধারণ করা হয়।
* **Cascading:** এর অর্থ হলো স্টাইলগুলো উপর থেকে নিচে ক্রমানুসারে (Priority অনুযায়ী) অ্যাপ্লাই হয়।

> 💡 **সহজ উদাহরণ:** HTML যদি একটি বাড়ির ইটের গাঁথুনি বা কঙ্কাল হয়, তবে **CSS** হলো সেই বাড়ির দেয়ালের রঙ, টাইলস, ফিটিংস এবং ফার্নিচারের সাজসজ্জা। এটি ছাড়া একটি ওয়েবসাইট দেখতে অত্যন্ত সাদামাটা লাগে।

## 📌 CSS কেন ব্যবহার করা হয়? (Why do we use HTML/CSS?)
একটি ওয়েবসাইটকে ব্যবহারকারী বান্ধব এবং আকর্ষণীয় করতে CSS ব্যবহার করা বাধ্যতামূলক। এর মূল কারণগুলো হলো:

* **ডিজাইন ও কালার:** টেক্সটের কালার, ব্যাকগ্রাউন্ড কালার এবং ফন্টের সাইজ পরিবর্তন করার জন্য।
* **লেআউট তৈরি:** ওয়েবসাইটের কোন উপাদানটি ডানে থাকবে আর কোনটি বামে থাকবে (Layout Positioning) তা ঠিক করতে।
* **রেসপনসিভ ডিজাইন:** ওয়েবসাইট যেন মোবাইল, ট্যাবলেট এবং কম্পিউটার সব স্ক্রিনেই সুন্দরভাবে দেখা যায় (Media Queries)।
* **সময় বাঁচানো:** একটিমাত্র CSS ফাইল দিয়ে পুরো ওয়েবসাইটের শত শত পেজের ডিজাইন একবারে নিয়ন্ত্রণ করা সম্ভব।

## 📌 CSS কীভাবে কাজ করে? (How does CSS work?)
CSS মূলত তিনটি জিনিসের সমন্বয়ে কাজ করে: **Selector**, **Property**, এবং **Value**।



### ১. CSS এর লেখার নিয়ম (Syntax)
```css
selector {
    property: value;
}
```

* **Selector:** HTML এলিমেন্টটিকে বেছে নেওয়া হয় (যেমন: `p`, `h1`, `div`)।
* **Property:** এলিমেন্টের কোন বৈশিষ্ট্য পরিবর্তন করা হবে (যেমন: `color`, `font-size`)।
* **Value:** বৈশিষ্ট্যটির মান কী হবে (যেমন: `blue`, `50px`)।

### ২. CSS লেখার তিনটি পদ্ধতি (Three Ways to Apply CSS)
CSS লেখার মূলত তিনটি পদ্ধতি রয়েছে:

#### ১. ইনলাইন সিএসএস (Inline CSS)
HTML এলিমেন্টের `<style>` ট্যাগের ভেতরে CSS কোড লেখা হয়।

**উদাহরণ:**
```html
<h1 style="color: blue; font-size: 30px;">Welcome</h1>
```
**সুবিধা:** ছোট কোডের জন্য সহজ।
**অসুবিধা:** কোড বারবার লিখতে হয়, তাই বড় প্রজেক্টের জন্য উপযুক্ত নয়।

#### ২. ইন্টারনাল সিএসএস (Internal CSS)
HTML ফাইলের `<head>` সেকশনে `<style>` ট্যাগ ব্যবহার করে CSS লেখা হয়।

**উদাহরণ:**
```html
<head>
    <style>
        h1 {
            color: red;
        }
    </style>
</head>
```
**সুবিধা:** একটি পেজের জন্য যথেষ্ট।
**অসুবিধা:** অন্য পেজে ব্যবহার করা যায় না।

#### ৩. এক্সটারনাল সিএসএস (External CSS)
একটি সম্পূর্ণ আলাদা `.css` ফাইল তৈরি করে সেখানে CSS লেখা হয় এবং সেটিকে HTML ফাইলের সাথে লিংক করা হয়। এটিই সবচেয়ে বেশি ব্যবহৃত পদ্ধতি।

**উদাহরণ:**
**HTML ফাইল:**
```html
<head>
    <link rel="stylesheet" href="styles.css">
</head>
```
**styles.css ফাইল:**
```css
h1 {
    color: green;
}
```
**সুবিধা:** পুরো ওয়েবসাইটের ডিজাইন নিয়ন্ত্রণ করা যায় এবং কোড ক্লিন থাকে।
