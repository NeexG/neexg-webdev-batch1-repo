# 📘 HTML Basics


## 📌 HTML কী? (What is HTML?)
**HTML**-এর পূর্ণরূপ হলো **HyperText Markup Language**।

* **HyperText:** এর অর্থ হলো এমন কিছু টেক্সট, যা অন্য কোনো টেক্সট বা পেজের সাথে যুক্ত (**Hyperlink**) থাকে। অর্থাৎ, একটি লিংকে ক্লিক করলে যখন আরেকটি পেজ খোলে, সেটাই হাইপারটেক্সট।
* **Markup Language:** এটি কোনো প্রোগ্রামিং ল্যাঙ্গুয়েজ নয়, এটি একটি মার্কআপ ল্যাঙ্গুয়েজ। এর কাজ হলো কিছু নির্দিষ্ট **ট্যাগ (Tags)** (যেমন: `<h1>`, `<p>`, `<img>`) ব্যবহার করে একটি ওয়েবসাইটের ভেতরের বিভিন্ন উপাদান বা কনটেন্টকে চিহ্নিত (**Mark**) করা।

> 💡 **সহজ উদাহরণ:** একটি বাড়ি তৈরি করার সময় সবার আগে ইটের গাঁথুনি দিয়ে যে মূল কাঠামো বা কঙ্কাল (**Skeleton**) তৈরি করা হয়, একটি ওয়েবসাইটের ক্ষেত্রে HTML হলো সেই কাঠামো।

## 📌 HTML কেন ব্যবহার করা হয়? (Why do we use HTML?)
ওয়েব ডেভেলপমেন্টের দুনিয়ায় HTML হলো একদম প্রথম এবং বাধ্যতামূলক ধাপ। এটি ছাড়া কোনো ওয়েবসাইট তৈরি করা অসম্ভব। এর মূল কারণগুলো হলো:

* **ওয়েবসাইটের কঙ্কাল বা ভিত্তি তৈরি করতে:** একটা ব্রাউজার (যেমন: Chrome বা Firefox) নিজে থেকে বোঝে না কোনটা হেডিং, কোনটা প্যারাগ্রাফ বা কোনটা ছবি। HTML ব্রাউজারকে বলে দেয় যে, *"এটি একটি হেডিং"* বা *"এটি একটি বাটন"*।
* **কনটেন্ট সাজাতে:** আমরা ওয়েবসাইটে যে টেক্সট, অডিও, ভিডিও, ছবি, টেবিল বা ফর্ম দেখি—এই সবকিছুকে সঠিক জায়গায় বসানোর জন্য HTML ব্যবহার করা হয়।
* **ব্রাউজারের সাথে যোগাযোগ করতে:** সার্চ ইঞ্জিনগুলো (যেমন: Google) HTML কোড পড়ে বুঝতে পারে তোমার ওয়েবসাইটে কী তথ্য আছে, যা **এসইও (SEO)** বা সার্চে আসার জন্য অত্যন্ত জরুরি।

## 📌 HTML কীভাবে কাজ করে? (How does HTML work?)
HTML মূলত **ট্যাগ (Tags)** এবং **অ্যাট্রিবিউট (Attributes)**-এর মাধ্যমে কাজ করে। এর কার্যপ্রণালী নিচে দেওয়া হলো:

### ১. ট্যাগের মাধ্যমে ব্রাউজারকে নির্দেশ দেওয়া
HTML-এ সব কনটেন্টকে নির্দিষ্ট ট্যাগের ভেতরে রাখতে হয়। ব্রাউজার এই ট্যাগগুলো নিজে স্ক্রিনে দেখায় না, কিন্তু ট্যাগ অনুযায়ী কনটেন্ট সাজিয়ে দেখায়।
> **যেমন:** তুমি যদি লেখো `<h1>হ্যালো</h1>`, ব্রাউজার বুঝবে এটি একটি বড় হেডিং এবং স্ক্রিনে **"হ্যালো"** লেখাটি বড় করে দেখাবে।

### ২. ওয়েব পেজের স্ট্রাকচার তৈরি করা
একটি সাধারণ HTML পেজ একটি নির্দিষ্ট নিয়মে কাজ করে:
* `<html>` : পুরো পেজটিকে এর ভেতরে রাখতে হয়।
* `<head>` : এখানে ওয়েবসাইটের ভেতরের তথ্য বা মেটাডাটা থাকে (যা সরাসরি স্ক্রিনে দেখা যায় না, যেমন: টাইটেল)।
* `<body>` : এর ভেতরের অংশটুকুই ব্যবহারকারীরা স্ক্রিনে দেখতে পান (যেমন: লেখা, ছবি, বাটন)।

### ৩. CSS এবং JavaScript-এর সংযোগস্থল হিসেবে
HTML একাই সব কাজ করে না। এটি কেবল উপাদানগুলো স্ক্রিনে এনে রাখে। এরপর:
* **HTML** দিয়ে উপাদান আনা হয় (যেমন: একটি বাটন)।
* **CSS** দিয়ে সেই উপাদানকে সুন্দর করা হয় (যেমন: বাটনটির ব্যাকগ্রাউন্ড লাল করা)।
* **JavaScript** দিয়ে উপাদানটিকে সচল করা হয় (যেমন: বাটনে ক্লিক করলে একটি পপআপ মেসেজ আসবে)।
